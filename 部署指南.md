# WorldForge 快速部署指南 🚀

## 🎯 三种方式让你的网站上网！

### 方式一：局域网分享（最简单！5分钟搞定）⭐⭐⭐

#### 步骤：
1. **双击运行** `启动局域网分享.bat`
2. 在同一 Wi-Fi 下的手机/电脑访问显示的 IP 地址
3. 搞定！

**详细说明**：见 `QUICK_SHARE.md`

---

### 方式二：Vercel 部署（永久免费网址！）⭐⭐⭐⭐⭐

#### 步骤：
1. 安装 Git：https://git-scm.com/download/win
2. 注册 GitHub：https://github.com
3. 注册 Vercel：https://vercel.com
4. 推送到 GitHub，在 Vercel 导入
5. 配置环境变量 `ZHIPU_API_KEY`
6. 部署成功！获得永久网址！

**详细说明**：见 `DEPLOYMENT.md`

---

### 方式三：ngrok 公网（临时演示）⭐⭐⭐

#### 步骤：
1. 下载 ngrok：https://ngrok.com/download
2. 运行 `npm run dev`
3. 运行 `ngrok http 3000`
4. 获得公网网址分享给任何人！

**详细说明**：见 `QUICK_SHARE.md`
